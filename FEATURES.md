# QuickTrans 功能说明

## 📋 功能清单

### 1. 音频转录（语音识别）

**功能描述：** 将音频文件转换为文本

**支持格式：** MP3, WAV, M4A, FLAC, OGG

**支持语言：**
- 中文
- 英文
- 日文
- 韩文
- 以及其他 90+ 种语言

**技术实现：** Whisper (faster-whisper)

**离线支持：** ✅ 完全离线

**模型大小：**
- tiny: ~40 MB（最快）
- base: ~140 MB
- small: ~460 MB（推荐）
- medium: ~1.5 GB
- large: ~2.9 GB（最准确）

---

### 2. 多语言文本翻译

**功能描述：** 将文本从一种语言翻译成另一种语言

**支持的语言对：**

| 源语言 | 目标语言 | 离线 |
|---------|----------|------|
| 中文 (zh) | 英文 (en) | ✅ |
| 英文 (en) | 中文 (zh) | ✅ |
| 英文 (en) | 日文 (ja) | ✅ |
| 英文 (en) | 韩文 (ko) | ✅ |
| 英文 (en) | 法文 (fr) | ✅ |
| 英文 (en) | 德文 (de) | ✅ |
| 英文 (en) | 西班牙文 (es) | ✅ |
| 日文 (ja) | 英文 (en) | ✅ |
| 韩文 (ko) | 英文 (en) | ✅ |
| 法文 (fr) | 英文 (en) | ✅ |
| 德文 (de) | 英文 (en) | ✅ |
| 西班牙文 (es) | 英文 (en) | ✅ |

**技术实现：** MarianMT (Helsinki-NLP/opus-mt 系列)

**离线支持：** ✅ 完全离线

**模型大小：** 约 300 MB/语言对

**特点：**
- 首次使用自动下载模型
- 模型缓存在本地
- 后续使用无需联网
- 完全免费

---

### 3. 繁简转换

**功能描述：** 中文简体和繁体互相转换

**支持方向：**
- 简体 → 繁体
- 繁体 → 简体

**技术实现：** OpenCC

**离线支持：** ✅ 完全离线

---

## 🔄 工作流程

### 流程 1：音频转录

```
音频文件 (MP3/WAV)
    ↓
Whisper 模型
    ↓
文本输出
```

### 流程 2：文本翻译

```
输入文本 (中文)
    ↓
MarianMT 翻译模型
    ↓
翻译结果 (英文)
```

### 流程 3：音频转录 + 翻译

```
音频文件 (中文录音)
    ↓
Whisper 识别
    ↓
中文文本
    ↓
MarianMT 翻译
    ↓
英文文本
```

## 🎯 使用场景

```

3. 保存使用
```
